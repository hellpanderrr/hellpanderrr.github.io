About This Implementation

This Armenian transcription app uses the Wiktionary Armenian Pronunciation Module to generate phonetic/phonemic transcriptions for Armenian text.

The system processes Armenian text through a rule-based approach that handles both Western and Eastern Armenian pronunciation patterns.

Dialects Supported

  • Western - Western Armenian pronunciation patterns
  • Eastern - Eastern Armenian pronunciation patterns

Transcription Forms

  • Phonetic: A detailed, narrow transcription that shows subtle, non-distinctive variations in pronunciation (allophones). It aims to represent how a word is actually said.
  • Phonemic: A simplified, broad transcription that shows only the sounds that are essential for distinguishing meaning (phonemes). It represents the abstract sound system of the language.

Technical Information

This implementation is based on the Wiktionary Armenian pronunciation module, which provides comprehensive coverage of Armenian phonological rules and dialectal variations for both Western and Eastern Armenian.

For technical issues or suggestions, please visit our GitHub repository.
