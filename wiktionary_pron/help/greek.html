About This Implementation

This Ancient Greek transcription app uses the Wiktionary Ancient Greek Pronunciation Module to generate phonemic transcriptions for Ancient Greek text.

The system processes Ancient Greek text through a rule-based approach that handles various historical pronunciation patterns from different periods and regions.

Dialects Supported

  • 5th BCE Attic - Classical Attic Greek pronunciation (5th century BCE)
  • 1st CE Egyptian - Hellenistic Greek pronunciation in Egypt (1st century CE)
  • 4th CE Koine - Koine Greek pronunciation (4th century CE)
  • 10th CE Byzantine - Byzantine Greek pronunciation (10th century CE)
  • 15th CE Constantinopolitan - Late Byzantine pronunciation (15th century CE)

Transcription Forms

  • Phonemic: A simplified, broad transcription that shows only the sounds that are essential for distinguishing meaning (phonemes). It represents the abstract sound system of the language.

Technical Information

This implementation is based on the Wiktionary Ancient Greek pronunciation module, which provides comprehensive coverage of Ancient Greek phonological rules and historical pronunciation variants across different periods.

For technical issues or suggestions, please visit our GitHub repository.
