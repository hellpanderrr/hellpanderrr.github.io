About This Implementation

This IPA transcription tool uses Wiktionary pronunciation modules to generate phonetic and phonemic transcriptions for multiple languages. Each language implementation is based on the comprehensive phonological rules and patterns available in Wiktionary's Lua modules.

Each help page provides detailed information about:

  • Supported dialects and variants - Specific pronunciation patterns for different regions or time periods
  • Transcription forms - Phonetic vs. phonemic transcription options
  • Technical implementation - Details about the underlying Wiktionary module and any special processing
  • Usage instructions - Specific guidance for optimal results with that language
