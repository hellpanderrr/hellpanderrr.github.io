About This Implementation

This Latin transcription app uses the Wiktionary Latin Pronunciation Module to generate phonetic/phonemic transcriptions for Latin text.

The system processes Latin text through a rule-based approach that applies Classical Latin pronunciation patterns. It automatically handles macronization (adding long vowel marks) to ensure accurate pronunciation.

Dialects Supported

  • Classical - Traditional Classical Latin pronunciation as reconstructed by scholars
  • Ecclesiastical - Church Latin pronunciation used in religious contexts
  • Vulgar - Late Latin/Vulgar Latin pronunciation patterns

Transcription Forms

  • Phonetic: A detailed, narrow transcription that shows subtle, non-distinctive variations in pronunciation (allophones). It aims to represent how a word is actually said.
  • Phonemic: A simplified, broad transcription that shows only the sounds that are essential for distinguishing meaning (phonemes). It represents the abstract sound system of the language.

Technical Information

This implementation is based on the Wiktionary Latin pronunciation module, which provides comprehensive coverage of Latin phonological rules and historical pronunciation variants.

Macrons are added based on the predefined dictionary.
